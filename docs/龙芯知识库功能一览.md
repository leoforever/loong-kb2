# 龙芯知识库 — 功能一览

## 多库并行问答

一个提问同时并发检索所有已授权知识库，合并 top chunk 后一次 LLM 生成答案。流式输出，边生成边展示。

## 多模型支持（无缝切换）

LLM 和嵌入模型均支持多 provider 切换，修改 `config.yaml` 无需重启服务。

| 用途 | 支持的 Provider | 说明 |
|------|----------------|------|
| **LLM** | MiniMax（默认）、Qwen（本地/远程） | MiniMax：API 调用；Qwen：支持本地部署或远程服务 |
| **嵌入模型** | Ollama（默认）、SiliconFlow | Ollama：本地推理；SiliconFlow：API 调用 |

> 切换方式：修改 `config.yaml` 中的 `llm.provider` 和 `embedding.provider`，保存后自动生效。

## 知识库类型

| 类型 | 说明 | 依赖 |
|------|------|------|
| 通用分段 | 常规文档，段落级切分 | RAG-Server |
| 父子分段-全文 | 父子层级，保留完整上下文 | RAG-Server |
| 父子分段-段落 | 父子层级，段落独立检索 | RAG-Server |
| 本地问答库 | CSV 导入 FAQ，纯本地管理 | 无 |

## RBAC 权限体系（完全自定义）

支持创建任意角色和用户，每个角色可自由绑定指定的知识库。admin / developer / viewer 仅为初始示例角色，可按需删除或重建。

## RAG-Server 文档知识库

RAG-Server 负责文档上传、解析（PDF/DOCX/TXT）、分段落、嵌入存储和检索。Loong KB 通过 REST API 与 RAG-Server 交互。

## 本地问答库（重点功能）

纯本地管理，CSV 两列（问题,答案）批量导入或手动逐条添加。底层 FAISS 向量索引，导入时自动构建，支持一键重建。

## 技术栈

Flask · SQLite · FAISS · RAG-Server · MiniMax/Qwen LLM（流式输出）
